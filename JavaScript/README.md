# 尚硅谷 JavaScript 基础 & 实战

## p1 - JS 简介

- 什么是语言
  - 计算机是一个由人来控制的机器
  - 要学习的语言就是人和计算机交流的工具，人类通过编程语言来控制、操作计算机
  - 编程语言和中文、英文本质上没有区别，只是语法比较特殊
  - 语言的发展
    - 纸带机：机器语言 => `0 & 1`
    - 汇编语言：符号语言
    - 现代语言：高级语言
- 起源
  - JS 诞生于 1995 年，它的出现主要是用于处理网页中的前端验证
  - 前端验证：指检查用户输入的内容是否符合一定的规则。
  - 比如：用户名的长度，密码的长度，邮箱的格式
